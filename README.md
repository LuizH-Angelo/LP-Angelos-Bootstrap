# LP-Angelos-Bootstrap

> Projeto / Avaliação Final da matéria de Front End do segundo semestre da faculdade

## 📋 Descrição

Este projeto é uma Landing Page desenvolvida como trabalho final da disciplina de Front End, utilizando as tecnologias web fundamentais para criar uma página responsiva e moderna.

## 🛠️ Tecnologias Utilizadas

- **HTML** (67.7%) - Estrutura e marcação semântica
- **CSS** (29%) - Estilização e layout responsivo
- **JavaScript** (3.3%) - Interatividade e funcionalidades dinâmicas

## 📁 Estrutura do Projeto

```
LP-Angelos-Bootstrap/
├── index.html          # Página principal
├── styles.css          # Estilos da aplicação
├── script.js           # Scripts e interatividade
└── README.md           # Este arquivo
```

## 🚀 Como Usar

### Opção 1: Clonar o Repositório

```bash
git clone https://github.com/LuizH-Angelo/LP-Angelos-Bootstrap.git
cd LP-Angelos-Bootstrap
```

### Opção 2: Abrir Diretamente

1. Baixe os arquivos do repositório
2. Abra o arquivo `index.html` no seu navegador preferido

## 💻 Visualizar Online

Você pode visualizar o projeto abrindo o arquivo `index.html` diretamente no navegador.

## ✨ Funcionalidades
